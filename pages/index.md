
## 作品简介

《**Alpha Zone**》（中文名：**阿尔法战区**）是由 Ethernos Studio 开发的一款近未来核冬天背景生存射击游戏。

游戏设定在 **2028–2031** 年间，DeWorld 组织使用失踪核弹袭击俄罗斯东北部及阿拉斯加后，被 EPUN（地球联合国）划定为"无主权封锁区"的阿尔法地带。玩家将深入核废墟、极寒地带与百股流民武装横行的死亡区域，在资源匮乏与动态威胁中求生，并揭开 EFGH 研究机构与 SR 实验室遗留的秘密。

## 世界观概览

| 项目               | 内容                                                                             |
| :----------------- | :------------------------------------------------------------------------------- |
| **时间线**   | 2025 年阶段0溃败前夜 → 2028 年氢弹袭击 → 2031 年黑域日常                       |
| **核心区域** | 阿拉斯加迪纳科山辐射区、加拿大被占领土、俄罗斯东北部                             |
| **主要势力** | EPUN（封锁方）、DeWorld（地下净化者）、EFGH（原研究机构）、SR 实验室（深渊科技） |
| **生存法则** | 开火优先于说话，沉默优先于呼救                                                   |
| **特色机制** | 子弹货币化、装备反噬系统、动态威胁等级（1–5 级）、钥匙卡权限（Lv1–Lv4）        |
| **经济核心** | 三级货币体系（卢布 / 美元 / 小麦粉），年通胀 1300%                               |

## 开发时间轴

| 时间                   | 事件                                                               |
| :--------------------- | :----------------------------------------------------------------- |
| **2024 年**      | 项目立项，前身为《ESCAPE FROM ALPHA ZONE》                         |
